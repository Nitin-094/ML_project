If you want to run this file.Please do the following steps:-

1). Create a python environment.
	
	In terminal run following commands- 

		1) 'conda create -n environment_name python=3.10'
		2) 'activate environment_name'

2). In that environment install all of that mentioned in requirements.txt
		
		1) 'pip install -r requirements.txt'

3). Now run main.py from terminal.
		
		1) 'python main.py'
